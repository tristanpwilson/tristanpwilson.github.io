Web Development & Design

Design and development of a web portfolio for a freelance model.

2019

Design & Development of JayHagram.com

In 2019, I created a modern web portfolio for a friend of mine who works as a freelance model and goes by the alias Jay Hagram. I designed and developed the site myself using Bootstrap 4.3 as a basic framework. For "Portfolio" and "Latest" pages, I created a responsive grid-style design with modals that are dynamically populated with high resolution versions of the images via JS/jQuery. Using the Instagram API and the Instafeed.js library developed by Steven Schobert, I was able to automatically pull new photos posted to the model's Intagram and display them on the "Latest" page, with a somewhat seamless integration of the media into the existing site design. All pages on the site are fully accessible across a range of disabilities (adhering to WCAG 2.1 AA standards), and are continuously responsive on any screen size.
